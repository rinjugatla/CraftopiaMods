概要
　音の再生位置をカメラ位置からキャラクタの位置に変更します。
　この変更により、キャラクタよりも後ろから音が聞こえる現象が改善されます。
　プラグイン導入後初回のゲーム起動時に設定ファイルが作成されます。
　設定ファイル名: me.rin_jugatla.craftopia.mod.FixEars.cfg

設定
　IsForwardByPlayer: デフォルトfalse
　　true: キャラクタの方向と耳の方向が一致します。
　　false: カメラの方向と耳の方向が一致します。(こちらの方が使いやすそう)

Overview
 Changes the sound playback position from the camera position to the character position.
 This change will correct the problem of sound coming from behind the character.
 A configuration file will be created the first time you run the game after installing the plugin.
 Configuration file name: me.rin_jugatla.craftopia.mod.FixEars.cfg

Settings
 IsForwardByPlayer: default false
  true: The direction of the character matches the direction of the ears.
  false: The direction of the camera will match the direction of the ears. (This may be easier to use).
