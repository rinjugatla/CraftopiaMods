概要
　セーブデータフォルダ名、サーバ設定ファイル名を変更します。

　初回起動時に「BepInEx\config」フォルダ内に設定ファイル「me.rin_jugatla.craftopia.mod.ServerUtility.cfg」が自動作成されます。
　設定は設定ファイルの説明を参照して下さい。
　同梱の「sample - me.rin_jugatla.craftopia.mod.ServerUtility.cfg」は初期値の設定ファイルです。

Summary
　Changes the name of the saved data folder and server configuration file.

　The configuration file "me.rin_jugatla.craftopia.mod.ServerUtility.cfg" will be automatically created in the "BepInEx\config" folder at the first startup.
　Please refer to the description of the configuration file for the settings.
　The included "sample - me.rin_jugatla.craftopia.mod.ServerUtility.cfg" is the default configuration file.
